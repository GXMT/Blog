안녕하세요. 이 글은 블로그가 잘 동작하는지 확인하기 위한 첫 번째 샘플 포스트입니다.

## 이 블로그의 특징

- **제로 JS 정적 렌더링** — 빠른 로딩과 안정적인 Core Web Vitals
- **CLS 없는 광고 슬롯** — 광고가 로드돼도 본문이 밀리지 않음
- **전문(全文) RSS/Atom 피드** — 네이버·다음이 본문 전체를 수집
- **자동 sitemap + JSON-LD + Open Graph** — 구글 색인과 공유 미리보기 최적화

## 코드 블록 예시

```js
console.log('Hello, world!');
```

이제 `src/content/blog/` 폴더에 마크다운 파일을 추가하기만 하면 글이 자동으로 발행됩니다.
